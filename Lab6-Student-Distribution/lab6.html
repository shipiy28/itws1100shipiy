jQuery Quick Reference

  • $(selector).click(function) - Add click event handler
  • $(selector).text('new text') - Change text content
  • $(selector).css('property', 'value') - Change CSS
  • $(selector).addClass('className') - Add CSS class
  • $(selector).removeClass('className') - Remove CSS class
  • $(selector).toggleClass('className') - Toggle CSS class
  • $(selector).hide(duration) - Hide with animation
  • $(selector).show(duration) - Show with animation
  • $(selector).append('content') - Add content to end
  • $(document).on('event', 'selector', function) - Event delegation

Debugging Tips:

  • Open Developer Tools (F12) to see console messages
  • Use console.log() to debug your code
  • Check the Console tab for errors
  • Use $(selector).length to verify elements exist

Think About: Why don't dynamically added elements respond to click events?

üí° Learning Moment

This problem demonstrates a key concept in dynamic web development. Events are only attached to elements that exist when the page loads. New elements added later don't automatically inherit these event handlers.
